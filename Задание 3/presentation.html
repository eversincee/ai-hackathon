Стандартные методы

Допусковый контроль
Срабатывает когда отказ уже произошёл

Наше решение

Комбинация четырёх статистических детекторов с автокалибровкой:

  • Обнаруживает фактические отказы → Bad
  • Предупреждает до отказа → Potential
  • Без ложных срабатываний → Good

Обоснование подхода

Почему не машинное обучение?

  • Требует размеченных данных об отказах — их нет
  • Нейросети не помещаются в 256 КБ на канал
  • Невозможно обучить прямо на контроллере
  • Чёрный ящик — нельзя объяснить срабатывание

Почему не простые пороги?

  • Фиксированный порог не учитывает прогрев ГТД
  • Не обнаруживает медленный дрейф внутри допуска
  • Не отличает шум от залипания датчика
  • Требует ручной настройки под каждый тип датчика

Выбранный подход: комбинация лёгких статистических детекторов

Темп

Скачок за один шаг — однозначный признак выброса или удара по датчику

Дисперсия

Нулевая дисперсия = залипание; аномально высокая = прибор шумит

CUSUM

Накапливает малые смещения, которые каждый по себе — норма, в сумме — отказ

EWMA

Сглаженное среднее выявляет нарастающий дрейф задолго до выхода за допуск

Калибровка

Первые 250 отсчётов (~5 сек). Метод Уэлфорда: вычисление среднего и σ без буфера. Отбрасывание выбросов >3σ.

Детектирование

O(1) на отсчёт для каждого детектора. Адаптивное опорное среднее следит за режимом ГТД без ложных тревог.

Вывод

Один CSV-файл на канал, формат совпадает с примером. CRLF-строки, CP1251, разделитель «;».

Адаптивное опорное среднее

Проблема

Температура масла ГТД растёт с 18°C до 54°C при прогреве — 98% флагов были Potential ложно.

Решение

Поле ref_mean адаптируется к рабочей точке двигателя со скоростью α = 0.02 (~50 шагов lag).


CUSUM и EWMA сравниваются с ref_mean, а не с калибровочным средним.

Результат

Наборы данных 2 и 4 — прогрев ГТД больше не вызывает ложных Potential.

/* Обновляем очень медленно */
ref_mean = α·value
         + (1-α)·ref_mean

/* α = 0.02 — lag ~50 шагов */

/* CUSUM сравнивает с ref_mean */
diff = value - ref_mean
S+ = max(0, S+·0.9 + diff - slack)
S- = max(0, S-·0.9 + slack - diff)

if S+ > threshold
    → Potential
