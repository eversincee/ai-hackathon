# ИИ — Автоматизация

Репозиторий с решениями хакатона РТУ МИРЭА 2026 «ИИ — Автоматизация».

## Задание 1 — Автоматизация оцифровки паспортов оборудования

Веб-система для автоматического извлечения данных из PDF-паспортов оборудования и интеграции с 1С.

Применяет ансамблевый подход: параллельное распознавание через VLM (Qwen2.5-VL) и OCR→LLM (Surya + Qwen2.5), с последующим голосованием и валидацией. Все модели работают локально через Ollama.

Стек: **Next.js · FastAPI · SQLite · Surya · Qwen2.5-VL · Ollama**

Подробнее: [Задание 1/README.md](Задание%201/README.md)

## Задание 3 — Диагностика сигналов датчиков

Модуль на C99 для предиктивного анализа сигналов аналоговых датчиков газотурбинного двигателя в реальном времени.

Самообучается на первых 250 отсчётах, затем четыре параллельных детектора помечают каждый отсчёт флагом: `0` Bad / `1` Good / `2` Potential. Поддерживает кросс-компиляцию под ARM и MIPS, не использует динамическую память.

Стек: **C99 · GCC · Make**
